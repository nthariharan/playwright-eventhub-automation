# Playwright EventHub Automation

End-to-end Playwright automation suite for [EventHub Practice Site](https://eventhub.rahulshettyacademy.com/login).

## Features
- 🔐 **Sign-in automation** with valid/invalid credentials
- 🎟️ **Event creation workflows** including form validations
- 🪑 **Seat booking flows** with confirmation checks
- 📊 **Assertions and reporting** for functional coverage

## Tech Stack
- [Playwright](https://playwright.dev/) for browser automation
- [Node.js](https://nodejs.org/en) for test execution
- GitHub Actions for CI/CD integration
